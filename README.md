# LLM Chat Frontend

## Overview

This is a frontend application to chat with an LLM, allowing users to perform CRUD operations on conversations.

## Technologies Used

- Node.js >=22
- Next.js 14
- React 18
- TypeScript
- Mantine 6
- React Query 5
- Docker

## Setup Instructions

### Prerequisites

- Node.js >=22
- npm

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/deepakchaula/llm-frontend.git
   cd llm-frontend
   npm install
   npm run dev
